# Food-Ordering-Chatbot

An AI-powered food ordering chatbot that enables users to place, modify, and track food orders through natural language conversations.

# Overview

- Supports food ordering, adding/removing items, and order tracking
- Uses intent classification to understand user requests
- Handles multi-turn conversations with context awareness
- Automates order management for a smooth user experience

# Tech Stack

- Python
- Natural Language Processing (NLP)
- Machine Learning
- MySQL (order storage & management)
- Dialogflow

# Key Features

- Intent detection (order, update, cancel)
- Entity extraction (food items, quantity)
- Dynamic order updates in real time
- Context handling for conversational flow
